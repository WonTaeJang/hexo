# Python Syntax 2
## 변수(Variables)

python에서는 값을 할당하면 변수가 생성됩니다. 

### example
``` python
x = 5
y = "Hello, World!"
```

파이썬에서는 변수를 선언하는 명령이 없습니다. 

Python Variable 에서 변수에 대해 자세히 알아보겠습니다. 

## 주석(Comments)
주석은 #으로 시작하고 해당 줄은 코드에 영향을 주지 않습니다.

### example
``` python
#This is a comment.
print("Hello, World!")
```
