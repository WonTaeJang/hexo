# 문자열 연결 (String Concatenation)
두 문자열을 연결하거나 결합하려면 + 연산자를 사용할 수 있습니다. 

### example
``` python
a = "Hello"
b = "World"
c = a + b
print(c)
```

### example
``` python
a = "Hello"
b = "World"
c = a + " " + b
print(c)
```
